## Angular
### Installation
`npm install`

### Run
`ng serve`

### Host
http://localhost:4200/

## NodeJS server
### Installation
`npm install --only=dev`

### Run
`mongod`  <br />
`npm run start`

### RESTful API
* http://localhost:4300/champions
* http://localhost:4300/champions/:championId
